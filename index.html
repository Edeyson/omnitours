Instructivo para Editar y Actualizar el Archivo planes1.json

Paso 1: Editar el JSON

El archivo planes1.json contiene los planes turísticos. Usaremos una herramienta interactiva para realizar los cambios:

  1. Accede a la herramienta de edición y abre el archivo planes1.json (Entrar a editar el archivo aquí).
  2. Agrega, elimina o edita los planes existentes, marcando o desmarcando los meses activos.
  3. Una vez realizados los cambios, copia el nuevo JSON generado.

Paso 2: Abrir el Repositorio en VS Code Web

  1. Abre el repositorio en VS Code Web haciendo clic aquí.
  2. En la barra lateral izquierda, navega hasta encontrar el archivo planes1.json.

Paso 3: Actualizar el Archivo planes1.json

  1. Haz clic en el archivo planes1.json para abrirlo.
  2. Pega el nuevo JSON que copiaste en el Paso 1, reemplazando el contenido existente.
  3. Guarda el archivo con Ctrl + S (o Cmd + S en Mac).

Paso 4: Subir los Cambios a GitHub

  1. Haz clic en el icono de "Control de Código Fuente" en la barra lateral izquierda de VS Code.
  2. Escribe un mensaje de commit como "Actualización de planes turísticos Mayo".
  3. Haz clic en "Commit" y luego en "Push" para enviar los cambios a GitHub.

Resumen de Pasos

  • Editar el JSON en la herramienta web Entrar a editar el archivo aquí.
  • Abrir el repositorio en VS Code Web.
  • Pegar el nuevo JSON en planes1.json y guardarlo.
  • Hacer commit y enviar (push) los cambios a GitHub.
